
Listas em Python


As listas em Python são estruturas de dados que permitem armazenar uma coleção ordenada de elementos. Elas são muito versáteis e uma das estruturas de dados mais utilizadas na linguagem. Aqui estão algumas características importantes sobre as listas em Python:

Ordenadas: 
Os elementos em uma lista são mantidos em uma ordem específica, o que significa que você pode acessar os elementos da lista pela sua posição.

Mutáveis: 
As listas em Python são mutáveis, o que significa que você pode alterar, adicionar e remover elementos após a criação da lista.

Heterogêneas: 
As listas podem conter elementos de diferentes tipos de dados, como inteiros, strings, booleanos, outras listas, dicionários, entre outros.

Indexação: 
Você pode acessar elementos individuais em uma lista usando índices. O primeiro elemento tem índice 0, o segundo índice 1 e assim por diante. Além disso, você pode usar índices negativos para contar a partir do final da lista.

Comprimento Variável: 
As listas em Python podem crescer ou encolher dinamicamente para acomodar a adição ou remoção de elementos.

Métodos úteis: 
Python fornece uma variedade de métodos embutidos que facilitam a manipulação de listas, como append(), insert(), remove(), pop(), sort(), reverse(), entre outros.
